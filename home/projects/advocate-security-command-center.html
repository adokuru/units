Advocate Security Command Center

Confidential Location

The goal for the Advocate Security Command Center was to renovate an existing space and create a centralized security command center to serve Advocates 12 Hospitals and 220+ group offices. The space would be planned to allow the opportunity to support 24 hospitals. This would replace disparate individual public safety departments that were scattered between multiple hospitals, and establish the need for a unified design approach. The Advocate Security Command Center is a high-security environment ready for around-the-clock operation and showcases their new enterprise security program with workstations, a briefing room with visual access to the command center, and state of the art server room.
